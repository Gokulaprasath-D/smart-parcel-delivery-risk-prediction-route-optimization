The Smart Parcel Delivery System is a real-time AI-based platform that improves last-mile delivery using smart routing, live tracking, and automated WhatsApp communication. # 📦 Smart Parcel Delivery System 🚚



\## 🔍 Overview

The Smart Parcel Delivery System is a real-time AI-based platform designed to improve last-mile delivery efficiency. It combines intelligent routing, live tracking, and automated customer communication into a single system to reduce delivery failures and improve agent productivity.



\---



\## 🚀 Features

\- 📍 Real-time GPS tracking of delivery agents  

\- 📲 Automated WhatsApp notifications for customer availability  

\- 🧠 AI-based risk and priority prediction  

\- 🗺️ Single Active Route navigation (reduces confusion)  

\- 🔄 Dynamic route recalculation  

\- 🔊 Voice alerts for delivery updates  



\---



\## 🛠️ Tech Stack

\- \*\*Backend:\*\* FastAPI (Python)  

\- \*\*Frontend:\*\* React (Vite)  

\- \*\*Real-Time Communication:\*\* Socket.IO  

\- \*\*Maps \& Routing:\*\* OpenStreetMap (OSRM)  

\- \*\*Machine Learning:\*\* Scikit-learn
